# Calten

Turn any book into a test of understanding.

🌐 https://calten.io

---

## What is Calten?

Calten is an AI-powered tool that helps you generate questions from a PDF to improve retention through active recall.

Instead of passively reading, you can test yourself and actually remember what you read.

---

## How it works

1. Upload your PDF or book
2. AI analyzes the content
3. Questions are generated instantly
4. Test your understanding

---

## Why use Calten?

* Improve retention through active recall
* Learn faster from any book or document
* No manual effort — just upload and start
